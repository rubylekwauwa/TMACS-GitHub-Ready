# T-MACS Azure Static Website

This folder is the CAS-free static website package for Azure Storage Static Website.

Upload these files to the `$web` container in your Azure Storage account. Set both the index document name and error document path to `index.html` if you want direct links to return to the app shell.

Included files:

- `index.html`
- `app.js`
- `styles.css`
- `mobile-app.js`
- `mobile-home.js`
- `mobile-home.css`
- `mobile.css`
- `guillermo-valdes.png`
- `welcome-tour.mp3`

No Azure Functions, CAS callback, session cookies, or Static Web Apps configuration are included in this package.
